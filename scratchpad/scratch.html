Javascript

Operators

( = ) is an assignment operator, which assigns values to variables. ( == ) is a loose equality operator, which will attempt to convert and compare operands of different types. ( === ) is a strict equality operator that compares two operands and checks whether both value and type are equal. ( != ) is a loose inequality operator, and ( !== ) is a strict inequality operator.

Comparison operators (like those above and those to come) can return either true or false. The operator === means "equal value and equal type." The operator !== means "not equal value or not equal type." When using "greater (or lesser) than or equal" the operators >= and <= should be used, with the equals sign on the right-hand side.

( ! ) reverses the value of a boolean. ( && ) returns true iff (not a typo) the operands are true. ( || ) returns true if at least one of the operations is true.
